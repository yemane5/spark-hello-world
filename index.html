ChatGPT Desktop

Open App Web

Linear

Track tasks, issues, bugs, project progress, and team workflow.

Open App Web

GitHub Repo

View your remote repository, commits, files, and project history online.

Open Repo Profile
Use this to verify that your push succeeded.
Workflow
Recommended SPARK workflow
  1. Plan the work in ChatGPT.
  2. Create or review the task in Linear.
  3. Edit files in VS Code.
  4. Ask Codex for coding help when needed.
  5. Commit changes in GitHub Desktop.
  6. Push to GitHub.
  7. Verify the files on GitHub.
  8. Update the Linear task status.
