SPARK Tools Dashboard

Quick guide for when to use each tool.

ChatGPT Desktop

Use for planning, explanations, debugging help, summaries, and general guidance.

Launch: Search Windows for ChatGPT

Codex

Use for AI-assisted coding inside a project/repository.

Launch: Search Windows for Codex

VS Code

Use to edit files like HTML, Markdown, Python, JavaScript, and AGENTS.md.

Launch: Search Windows for Visual Studio Code

GitHub

Use to store your code online, verify commits, and share your repository.

Open spark-hello-world repo

GitHub Desktop

Use to commit, push, pull, and manage Git changes without command-line commands.

Launch: Search Windows for GitHub Desktop

Linear

Use to manage project tasks, issues, bugs, and progress.

Open Linear Web

Desktop: Search Windows for Linear

Typical Workflow

Plan in ChatGPT → Track task in Linear → Code in VS Code/Codex → Commit in GitHub Desktop → Verify on GitHub.
